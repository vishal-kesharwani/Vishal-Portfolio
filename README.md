# Vishal Kesharwani Portfolio

Modern portfolio built with Next.js, React, Framer Motion, and Tailwind CSS.

## Owner

Vishal Kesharwani

## Live

- Production: [vishalkesharwani.in](https://vishalkesharwani.in)
- Preview: [Vercel deployment](https://vishal-portfolio-flax.vercel.app/)

## Highlights

- Cloud and DevOps-inspired hero and Cloud Lab section
- First-person project, internship, education, and achievements content
- Responsive design with subtle motion
- Contact form wired to `vishalkes1378@gmail.com`
- Custom domain support for GitHub Pages or Vercel

## Local Development

```bash
npm install
.\dev.cmd
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

This project is set up for static export.

- Keep `public/CNAME` for the custom domain
- Deploy the exported `out` folder to GitHub Pages or Vercel
- Update DNS records for `vishalkesharwani.in` based on the chosen host
